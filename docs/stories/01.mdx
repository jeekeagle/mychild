# 五只小鸭去游泳

✍️ 婧姝 · 2026-06-13


<div class="fairy-story-layout">
<div class="fairy-story-text">

从前有五只可爱的小鸭子，他们最喜欢的事情就是——游泳！

有一天，天气特别好，太阳公公挂在天上笑眯眯的。五只小鸭排着队，一个接一个地跳进水里。

“扑通！扑通！扑通！扑通！扑通！”

五声水响，五只小鸭都在水里游了起来。他们游啊游，游啊游，身上变得湿哒哒的。

游累了，小鸭们爬上岸，用毛巾把自己擦干，然后开开心心地回家了。

第二天，小鸭们又来游泳了。

可是——咦？奇怪的事情发生了！

昨天还是清清亮亮的水，今天怎么变成五颜六色的了？红的、黄的、蓝的，像彩虹掉进了水里一样。

小鸭们你看看我，我看看你，都不知道发生了什么。

"是谁把水弄成这样的呀？"小鸭们很着急。

就在这时候，他们看到岸边有一个人，正往水里倒一些奇怪的液体。那些液体流进水里，水就变成了彩色。

原来，是有人在污染河水！

小鸭们立刻去找警察叔叔，告诉了他们这件事。警察叔叔很快就把污染水的人抓起来了，河水又变回了清清亮亮的样子。

小鸭们又可以开心地游泳啦！
