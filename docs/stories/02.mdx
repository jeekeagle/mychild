export const ViewportSetter = () => {
  useEffect(() => {
    function setVp(){
      var w = window.innerWidth || document.documentElement.clientWidth;
      document.body.setAttribute('data-viewport', w < 996 ? 'mobile' : 'desktop');
    }
    setVp();
    window.addEventListener('resize', setVp, {passive: true});
    window.addEventListener('orientationchange', setVp);
    return () => {
      window.removeEventListener('resize', setVp);
      window.removeEventListener('orientationchange', setVp);
    };
  }, []);
  return null;
};

# 公主与王子

从前有个小公主，她穿着粉色的裙子，最喜欢去探险了。她会在城堡的花园里找蝴蝶，在森林里捡漂亮的石头，在河边看小鱼游来游去。

有一天，公主在花园里遇到了一只毛茸茸的小狗。小狗摇着尾巴，汪汪叫着跑过来。公主蹲下来摸摸它的头，小狗就舔了舔她的手。他们很快就成了好朋友。公主给小狗取名叫"球球"，因为它的身体圆滚滚的，像个毛球一样。公主和小狗每天一起玩，一起跑，可开心了。

后来，公主又遇到了很多新朋友。有其他穿着漂亮裙子的小公主，有穿着披风的勇敢超人，还有更多可爱的小狗。大家虽然长得不一样，有的高，有的矮，有的会飞，有的只会跑，但都愿意一起玩。公主说："我们搭个秘密基地吧，每天都在一起玩！"大家觉得这个主意太棒了。他们找了一棵大大的树，在树下搭起了秘密基地，用树叶做屋顶，用石头做椅子。每天放学后，大家都跑到秘密基地里玩，有时玩过家家，有时玩捉迷藏，有时一起分享小零食。

有一天，公主在秘密基地附近遇到了一个特别的朋友。他是个勇敢的小男孩，穿着蓝色的衣服，就像故事里的王子一样。王子也喜欢探险，也喜欢交朋友。他看到公主和小狗在玩，就走过来问："我可以和你们一起玩吗？"公主笑着说："当然可以！"王子就加入了他们的游戏。公主和王子一起玩，一起笑，就像其他好朋友一样。王子会帮公主爬树摘果子，公主会帮王子找躲藏的地方，小狗会在旁边开心地跑来跑去。

他们每天都在一起玩，慢慢地，公主和王子发现他们最喜欢和对方在一起。王子会给公主摘最漂亮的花，公主会帮王子整理衣服。他们一起看星星，一起看日出，一起经历了很多很多开心的事情。

后来，公主和王子结婚了。他们举行了盛大的婚礼，所有的好朋友都来参加。小狗球球也穿上了小西装，开心地摇着尾巴。婚礼上，大家都为他们鼓掌，祝福他们永远幸福。

公主和王子幸福地生活在了一起。他们在城堡里建了一个大大的秘密基地，每天都有很多朋友来玩。再后来，他们生了一对可爱的双胞胎宝宝，一个像公主一样漂亮，一个像王子一样勇敢。一家人在一起，每天都开开心心，就像童话故事里那样，永远幸福地生活在一起。
