<!-- 移动端/桌面端检测：把结果写到 body[data-viewport]，
     CSS 用属性选择器控制（不依赖 @media，老浏览器 100% 生效） -->
export const ViewportSetter = () => {
  const [windowSize, setWindowSize] = useState('ssr');
  useEffect(() => {
    function update() {
      const w = window.innerWidth || document.documentElement.clientWidth;
      setWindowSize(w < 996 ? 'mobile' : 'desktop');
    }
    update();
    window.addEventListener('resize', update, {passive: true});
    window.addEventListener('orientationchange', update);
    return () => {
      window.removeEventListener('resize', update);
      window.removeEventListener('orientationchange', update);
    };
  }, []);
  useEffect(() => {
    if (windowSize === 'ssr') return;
    document.body.setAttribute('data-viewport', windowSize);
  }, [windowSize]);
  return null;
};

**小朵朵的勇敢探索**

从前有个叫小朵朵的小女孩，她很害羞，连超人都不怕的事情她都害怕。她最喜欢躲在自己的小房子里，不敢出来玩。

外面的世界看起来好大好陌生。小朵朵常常趴在窗户上往外看，看见外面的小朋友在草地上跑来跑去，笑声传得远远的。可她心里想："万一外面有可怕的东西怎么办？万一大家不喜欢我怎么办？"想到这里，她又缩回了小房子里。

有一天，小朵朵鼓起勇气，小心翼翼地走出了小房子。她的心跳得很快，手心都有点出汗了。她一步一步慢慢走，走到了草地边上。

她发现，草地软软的，踩上去很舒服。阳光暖暖地照在身上，还有小鸟在树上唱歌。原来外面没有她想象的那么可怕！

这时，几个小朋友跑了过来。"你好呀，要不要和我们一起玩？"一个扎着小辫子的女孩笑着问。

小朵朵有点紧张，但她点了点头。

小朋友们拉着小朵朵的手，一起玩起了捉迷藏。小朵朵躲在树后面，听见大家找她的声音，忍不住笑了起来。她发现，原来和大家一起玩这么开心！

后来，小朵朵每天都出来玩。她和朋友们一起吃苹果，一起玩游戏，一起看《小马宝莉》的故事书。有时候，他们会围坐在一起，轮流讲故事。小朵朵讲了一个关于小猫咪的故事，大家都听得津津有味。

慢慢地，小朵朵的朋友越来越多，数都数不过来了。连邻居家的小猫咪都跑过来听他们讲故事，森林里的小兔子、小松鼠也成了她的好朋友。

现在的小朵朵再也不害羞了，她每天都开开心心地和朋友们一起玩耍、一起探索这个奇妙的世界。她发现，只要勇敢尝试，就会遇到好多好多美好的事情！
