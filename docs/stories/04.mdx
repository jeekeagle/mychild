import {useEffect} from 'react';

{/* 移动端/桌面端检测：设置 body[data-viewport]，CSS 用属性选择器控制（不依赖 @media，新老浏览器 100% 生效） */}
export const ViewportSetter = () => {
  useEffect(() => {
    function setVp(){
      var w = window.innerWidth || document.documentElement.clientWidth;
      document.body.setAttribute('data-viewport', w < 996 ? 'mobile' : 'desktop');
    }
    setVp();
    window.addEventListener('resize', setVp, {passive: true});
    window.addEventListener('orientationchange', setVp);
    return () => {
      window.removeEventListener('resize', setVp);
      window.removeEventListener('orientationchange', setVp);
    };
  }, []);
  return null;
};

# 怪兽总动员

在一片神奇的森林里，住着一群可爱的小怪兽。这些小怪兽都长得一模一样的。

有一天，大家正在玩滑梯，突然发现一个奇怪的小怪兽站在旁边。他长得有点不一样，颜色也不一样。

小怪兽们好奇地问：“你是谁呀？”

新来的小怪兽说：“我是从很远的国家来的小朋友，我想学你们中国的东西。”

小怪兽们开心地说：“好呀好呀！来，我们教你，在中国，我们是这样说的——你好！”

新来的小怪兽认真地学：“你好……你好……我学会了！中国是用这个语言说的，真有意思。”

然后，他说：“我还有很多朋友，他们也是从世界各地来的。我把他们都带来好不好？这样我们的世界就更加丰富了，是不是？”

大家拍着手说：“太棒了！这样的话，不管小怪兽长什么样、说什么话，大家都可以在一起玩，世界就应该丰富呀！”

新来的小怪兽说：“对，今天是最美好的一天。我是来这里旅行的，希望你们能开心。”

小怪兽们说：“来吧，我们一起去更多地方玩！那不就是更加丰富的世界吗？”

于是，他们一起玩了跷跷板、滑梯、跳绳，在游乐场跑来跑去，最后还去了新朋友的家。他们最喜欢的地方终于建成了——一个所有小怪兽都能一起玩的家。

天黑了，新来的小怪兽说：“我学会中国话之后，以后就要这样说了，对不对？”

大家说：“对的！你学会了就要用。而且我们也可以学你的语言呀，这样我们就能互相听懂了，是不是？”

所有小怪兽都笑了。他们发现，原来世界这么好玩，就是因为每个人都不一样，但大家愿意一起玩，愿意学对方的话，愿意成为朋友。

这就是最好的世界。
